Inheritance

Inheritance is a key concept in Object-Oriented Programming (OOP) where a new class (called a child or subclass) takes on properties and behaviors (methods) of an existing class (called a parent or superclass).

This lets you reuse code and create more specific versions of general classes without rewriting everything.
